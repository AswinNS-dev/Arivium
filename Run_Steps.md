# Running Arivium Frontend

## Prerequisites
Ensure you have Node.js (v18+ recommended) and npm installed on your machine.

## Steps

1. **Navigate to the Project Directory**
   Ensure you are in the `Arivium` root directory.

2. **Install Dependencies**
   If you haven't already, install the necessary dependencies:
   ```bash
   npm install
   ```

3. **Start Development Server**
   Start the Vite development server:
   ```bash
   npm run dev
   ```
   This will run the application locally (typically at `http://localhost:5173`).

4. **Build for Production**
   To create a production build:
   ```bash
   npm run build
   ```

5. **Preview Production Build**
   To preview the built production app locally:
   ```bash
   npm run preview
   ```
